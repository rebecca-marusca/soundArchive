# soundArchive

An interactive soundboard I built to play around with the p5.js library and get some coding practice in.

Features include the background changing color randomly on each click, and a press and ripple animations for clicking the buttons.

This was pretty fun to build and fun to play around with shapes, sounds and animations. Definitely feels like something that would be cool in the 2010s era:))
